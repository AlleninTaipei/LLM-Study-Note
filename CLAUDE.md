# LLM-Study-Note

## Repository Purpose

A curated collection of educational reference notes on LLMs and AI engineering topics. This is a pure documentation repository — no code, build system, tests, or dependencies.

## Structure

- **Root directory** — 20 markdown study note files
- **`assets/`** — PNG images referenced inline within the markdown files
- **`README.md`** — index table listing all notes with titles and brief descriptions

## Content Conventions

- **File naming**: Descriptive title-case names (e.g., `Prompt Engineering.md`, `LLM Parameters and Memory Estimation.md`)
- **Images**: Stored in `assets/` and referenced in markdown with relative paths (`assets/filename.png`)
- **README table**: Each note appears as a row with its title and a short description; keep this table updated when adding new files
